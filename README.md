# Start the BackEnd

First change directory to the Server folder <br>
`cd Server`<br>
Install the dependencies<br>
`npm install`<br>
Run the server<br>
`node server`<br>

# Start the FrontEnd<br>

First change the directory to the Client folder<br>
`cd Client`<br>
Install the dependencies<br>
`npm install`<br>
Run the server<br>
`npm start`<br>
